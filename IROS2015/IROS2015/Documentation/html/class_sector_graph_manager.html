Detailed Description

A manager class to handle different instances of grid for different sectors. Each 'type' in the simulation may have a different grid object (layer) assigned to it, in order to calculate traffic congestion differently and assign different costs to different UAV traffic types. This class stores the layer and manages access based on type.
